# Assignment: crud-php-pdo-simple

এই এসাইনমেন্ট এর জন্য HTML, CSS, JavaScript, PHP, MySQL ব্যবহার করে তৈরি করা হয়েছে| এখানে কিছু ভুল কোড বা ভিন্ন কোড করা হয়েছে, আপনাদের কাজ হবে এই কোড ব্যবহার করে ভুল গুলো সমাধাণ করা এবং নিচের বর্ণনা অনুযায়ী এসাইনমেন্ট করা। আপনাকে HTML, CSS, JavaScript, PHP, MySQL যা যা শেখানো হয়েছে, শুধু মাত্র সেগুলো ব্যবহার করেই এসাইনমেন্ট করতে হবে । 
অন্যথায় আপনার মার্ক এর উপর প্রভাব পড়তে পারে । 

## এসাইনমেন্ট এ আপনাকে যা যা করতে হবেঃ

✓ database.sql file অনুযায়ী ডাটাবেজ তৈরি করতে হবে। এখানে টেবিল এর নাম হবে `employees` এর কলাম হবে `id` (PRIMARY KEY), `name`, `phone` (UNIQUE), `designation` (NULL) 

✓ সকল Page এ CSS (external file: `assets/app.css`) দিয়ে সকল content কে Horizontal & Vertical উভয় দিকে center করতে হবে।

✓ Home page এ `employees` এর লিস্টের টেবিল থাকবে। টেবিলে ID, Name, Phone, Designation দেখাবে। যাকে শেষে অ্যাড করা হবে তার তথ্য উপরে থাকবে।

✓ যেহেতু টেবিলে দেখানোর জন্যে আপনাকে কোন প্রকার ডেটা দেয়া হয় নি, সেহেতু আপনাকে নিজেকেই অন্তত ১০ জন ডেটা অ্যাড করতে হবে।

✓ ডাটা অ্যাড করার সময় প্রয়জনীয় চেন্জ এবং ভেলিডেশন করতে হবে। ডাটা অবশ্যই পোস্ট মেথডে হবে।

✓ ডাটা এডিট করার সময় প্রয়জনীয় চেন্জ এবং ভেলিডেশন করতে হবে। ডাটা অবশ্যই পোস্ট মেথডে হবে।

✓ ডাটা ডিলিট করার সময় প্রয়জনীয় চেন্জ এবং ভেলিডেশন করতে হবে।

✓ ডাটা অ্যাড, এডিট, ডিলিট করার সময় SQL Injection এর ঝুঁকি যেন না থাকে।

✓ সকল form এ JS (external file: `assets/app.js`) দিয়ে validation করা।

✓ সকল Page এ CSS (external file: `assets/app.css`) দিয়ে ডিজাইন সুন্দর করা, যেখানে Brand কালার কোড: `#5527ff`
